# Grocery Shop Management System

A Python-based system to manage a grocery shop, including inventory management and billing.

## Features
- Add, update, and delete products
- Track stock levels
- Generate bills with tax calculation
- Interactive console interface

## Files
- `main.py` — Main program
- `inventory.py` — Manage products and stock
- `billing.py` — Handle billing and receipts
- `config.py` — Settings like tax rate
